git

init        项目初始化
clone       从远程克隆一个项目到本地
status      查看当前库的状态
reset       将代码重置
add         将文件添加到暂存区
commit      将修改提交到本地仓库
push        将本地仓库的内容推送到远程仓库
pull        将远程仓库内容拉取到本地库
branch      分支管理
log         查看分支历史
merge       合并分支
remote      远程管理
diff        差异对比
